Parágrafos e quebras de linha

Para fazer um parágrafo use a tag -> <p> e </p>.

Para quebrar linhas,
desse jeito, use -> <br>.

Vamos adicionar símbolos especiais: Marca registrada ®, Copyright ©, Trademark ™, Euro €, Libra £ Yen ¥, Centavo Americano ¢, Delta Δ, Seta pra cima ↑, etc.

Vamos adicionar emojis: 🖖 🤓
